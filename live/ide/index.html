Việc nhập xuất dữ liệu

  1. Nhập dữ liệu qua STD Input
    • Dữ liệu được nhập thông qua các lệnh hỗ trợ đọc dữ liệu từ bàn phím do người dùng nhập như:
      • Read, ReadLn của Pascal
      • gets, scanf của C
      • cint >> ... của C++
      • input, sys.stdin của Python
      • System.Console.ReadLine(), System.Console.Read() và System.Console.ReadKey() của C#
      • ...
    • Khi chạy chương trình các dữ liệu đầu vào sẽ được gởi thông qua STD Input vì thế bạn cần nhập dữ liệu nào vào chương trình thì chỉ cần viết nó ở cửa sổ SDT Input. Mỗi dữ liệu xuống dòng tương ứng với 1 dòng dữ liệu người dùng nhập và nhấn Enter.
  2. Xuất dữ liệu qua STD Output
    • Dữ liệu được xuất thông qua các lệnh hỗ trợ ghi dữ liệu ra màn hình như:
      • Write, WriteLn của Pascal
      • Puts, printf của C
      • cout << ... của C++
      • print của Python
      • System.Console.Write(), System.Console.WriteLine() của C#
    • Dữ liệu được xuất ra khi chạy chương trình sẽ được hiển thị ở cửa sổ STD Output
  3. Xuất lỗi qua STD Error
    • Dữ liệu lỗi được xuất thông qua các lệnh hỗ trợ xuất lỗi như:
      • Writeln(StdErr , ...); của Pascal
      • cerr << ... , clog << ...của C++
    • Dữ liệu lỗi được hiển thị ở cửa sổ STD Error khi chạy chương trình
